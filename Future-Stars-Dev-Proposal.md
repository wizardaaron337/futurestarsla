## The Website — What's Been Built

A fully custom operations portal at **futurestarsla.com** with:

| Feature | What it does |
|---------|-------------|
| **PIN-based login** | Role-based access for all staff — JR, Lane, PJ, Caleb, Marlon |
| **Dashboard** | Live stats, quick links, org overview |
| **Inventory** | Barcode scanning, size breakdowns, baseball & soccer jerseys |
| **Tournament Manager** | Scheduling, lock/unlock, baseball/soccer filters, stats |
| **Trip Planner** | Calendar view, trip cards, upcoming & past trips |
